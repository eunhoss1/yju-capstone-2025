Logi-Optimus

AI 기반 배송-적재 E2E 최적화 시스템

물류 현장의 '경험'에 의존하던 비효율을 AI로 해결합니다.
최적의 경로와 적재 순서를 동시에 제공하여 시간을 단축하고 업무를 표준화합니다.

물류 현장의 '끊어진 고리'

현재 현장에서는 '경로 계산'과 '화물 적재'가 완전히 분리되어, 기사의 '경험'과 '감'에 의존합니다. 이는 비효율과 직결됩니다.

🚚

경로 최적화 (TSP)

지도 API가 최적의 배송 순서(A → C → B)를 알려줍니다.

👇 ❌
🧠

수동 적재 (경험)

경로의 역순(B → C → A)으로 싣는 것을 기사가 직접 머릿속으로 계산해야 합니다.

우리의 해결책: 'LIFO' 연동

Logi-Optimus는 이 '끊어진 고리'를 AI로 연결합니다.
경로가 계산되는 즉시, '배송 순서의 역순(LIFO)'을 핵심 제약 조건으로 변환하여 3D 적재 알고리즘에 자동 적용합니다.

시장은 비어있습니다 (경쟁 분석)

기존 솔루션은 '경로'만 제공하거나, '배송 순서(LIFO)'를 고려하지 않는 '적재'만 제공합니다. Logi-Optimus는 이 둘을 연동하는 유일한 경량 웹 솔루션입니다.

4단계 작동 방식

사용자는 파일만 업로드하면, AI가 모든 것을 계산하여 2개의 핵심 결과물(지도, 3D 뷰)을 제공합니다.

1

데이터 입력

주소와 물품 리스트 (크기, 무게 등) 엑셀 업로드

2

경로 계산 (TSP)

지도 API가 최적의 배송 순서(A→C→B)를 계산

3

AI 적재 계산 (LIFO)

경로의 역순(B→C→A)과 무게/파손을 고려한 AI 알고리즘 실행

4

결과 시각화

'2D 지도'와 '3D 적재 가이드'를 웹 화면에 동시 제공

현실을 반영한 AI 로직

단순 LIFO가 아닌, 교수님의 피드백을 반영한 현실적인 4가지 우선순위 룰을 적용합니다.

P0: 묶음 처리

동일 배송지 물품은 하나의 그룹으로 묶습니다.

P1: 무게/파손

무거운 것은 아래에, 파손주의는 위에 배치합니다.

P2: LIFO 순서

P1을 만족하는 선에서 배송 역순으로 안쪽부터 배치합니다.

P3: 공간 최적화

빈 공간을 최소화하는 3D 테트리스로 최종 위치를 확정합니다.

  • 적재 시간 단축

박영민 기사 (34세)

NEEDS

  • 가장 빠른 배송 순서(경로) 제공
